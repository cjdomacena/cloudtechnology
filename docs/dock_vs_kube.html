Docker

Docker is a publicly available containerization platform that provides software to ensure that the application under development can run in any environment. The Docker container is what holds a running instance of a Docker image. It contains everything needed to run the application that is created from the image. Docker containers can run the Docker image regardless of the host machineâ€™s OS.

Kubernetes

Kubernetes allows containers to run on multiple nodes. The networking, security, and scaling of a container can be automated across all nodes via a single command line. There is a master node that controls the scheduling of pods across various worker nodes.

For additional information on containers and virtual machines, check out Docker's website or Kubernetes' website.
